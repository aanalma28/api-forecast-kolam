Quick Start

Health Check
GET /api/health
Generate Forecast
POST /api/forecast
{
  "start_date": "2024-01-01",
  "periods": 30
}
Model Information
GET /api/model/info
Batch Forecasting
POST /api/forecast/batch
{
  "requests": [
    {"start_date": "2024-01-01", "periods": 15},
    {"start_date": "2024-02-01", "periods": 30}
  ]
}
